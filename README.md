# Portfolio

A software developer portfolio built with Next.js (App Router), TypeScript, and Tailwind CSS.

## Getting started

Requires **Node.js 20.9.0 or later**.

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deploy on Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/your-username/portfolio)

After your first deploy, add your live URL here as **Demo**.
